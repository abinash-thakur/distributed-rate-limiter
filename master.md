# 🚀 Distributed Rate Limiter (FAANG-Level System)

## 📌 Overview

A production-grade **distributed rate limiting system** designed for **high-scale APIs**, supporting:

* Multi-tenant hierarchical limits
* Control Plane + Data Plane architecture
* Multi-region deployment
* Adaptive throttling
* Real-time analytics
